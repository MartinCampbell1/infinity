# Architecture Diagrams

This directory contains committed topology diagrams for the Infinity control
plane.

- `infinity-topologies.html` - standalone HTML with inline SVG diagrams for
  local, staging, and production target topologies.

Open locally:

```bash
open /Users/martin/infinity/docs/architecture/infinity-topologies.html
```

The diagrams are static documentation. They do not replace live validation,
deployment manifests, or production readiness gates.
